# LEARNIX-school-system
LEARNIX
Smart Educational Performance Intelligence System

A data-driven platform designed to transform how educational performance is monitored, analyzed, and improved — from individual students to city-wide education systems.

Overview

LEARNIX is a centralized performance tracking and analytics platform that provides real-time insights into the education system at every level:

Student
Teacher
School
Wereda
Kifle Ketema
City (Addis Ababa)

It replaces fragmented, manual, and delayed reporting systems with a transparent, intelligent, and predictive system.

Core Objectives
Improve education quality through data
Enable real-time decision making
Increase transparency across schools
Detect performance issues early
Provide predictive insights for future outcomes
System Architecture (Hierarchy)
Level	Role	Access Scope
Teacher	Class-level insights	Individual students
Parent	Child performance	Their child only
School Admin	School-wide monitoring	All students & classes
Wereda	Multi-school oversight	Schools in wereda
Kifle Ketema	Regional monitoring	Weredas
Addis Ababa	City-wide intelligence	All kifle ketemas
